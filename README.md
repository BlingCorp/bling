## Features

- Layouts
    - mstab (master-slave tab layout)
    - centered
    - vertical
    - horizontal
    - equalarea
- Modules
    - Flash Focus
    - Tabbed container
    - Tiled Wallpaper
    - Wallpaper Easy Setup
    - Window Swallowing
    - Scratchpad
- Signals
    - Playerctl
- Widgets
    - Tag Preview
